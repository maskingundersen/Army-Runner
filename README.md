# Army Runner 🎮

A mobile-first army runner game built with Phaser.js.

## How to Play
- Swipe left and right to steer your army through gates
- Green gates are good, red gates are bad
- Grow your army and defeat the enemies at the end of each level!

## How to Host on GitHub Pages
1. Go to Settings → Pages
2. Set source to the main branch
3. Open the generated URL in Safari on your iPhone!
